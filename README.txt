Projeto - 
"Recriando a Interface do Netflix"

"Recrie a interface do principal site de streaming mundial utilizando tecnologias simples como HTML5, CSS3 e JavaScript. Nesse projeto você aprenderá: como estruturar um layout, técnicas de CSS3 com containers e variáveis, como posicionar os elementos com Flexbox e como utilizar plugins JQuery a favor da sua aplicação.

Dica: você pode dar um "fork" no repositório disponibilizado pelo expert para organizar melhor as suas alterações e evoluções, mantendo uma referência direta ao código original."

FEEDBACK
Conteúdo para desenvolvimento do Projeto: Show! 
Fácil aplicação, com pontos chaves ressaltados, porém um pouco incompleto.
Se a interface da Netflix se resume ao Banner Principal e ao Carrossel; está ótimo, porém sabemos que a interface da Netflix tem muito mais elementos.

Na interface da Netflix 

- ilustração (img/readme/01.png) - footer como fixar? como alinhar? responsividade? como colocar notificação? e numerar estas notificações?

- ilustração (img/readme/02.png) - pesquisar? como faz? é front-end? back-end?

- ilustração (img/readme/04.png) - compartilhar - fundo em meio/transparente, é uma nova página? ou é um estilo "pop-up" - pois dá para vê a página do filme atrás; então como faz isto?! E incluir links para compartilhar, não é apenas o link do perfil certo??? 


Enfim, dito tudo isto, é fato que o projeto tal como está trás muitos aprendizados, entretanto tratar como recriar plataforma da netflix, frusta a expectativa do desenvolvimento de uma aplicação "completa e muito dinâmica".

A Netflix como exemplo de qualidade de código é de fato excelente, porém sinto que não foi por completo abordado.
